# Housing Price Prediction

ML models predicting California house prices from real estate data.

## Overview

Comparison of Linear Regression vs Random Forest models on the California Housing dataset. Demonstrates regression, model evaluation, and hyperparameter tuning.

## Results

| Model | R² Score | MAE |
|-------|----------|-----|
| Linear Regression | 0.58 | $53,000 |
| Random Forest | 0.80 | $33,000 |

## Files

- `exploratory_analysis.ipynb` - Data exploration and visualization
- `model_optimization.py` - Model tuning experiments
- `breast_cancer_classifier.py` - Classification example

## Key Learnings

- Train/test split importance
- Cross-validation for model reliability
- Feature importance analysis
- Diminishing returns in model complexity
